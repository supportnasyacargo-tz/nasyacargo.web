# Nasya Cargo Website

## File Structure
```
nasya-website/
│── index.html       ← Home page
│── about.html       ← About Us page
│── services.html    ← Services page
│── track.html       ← Track Cargo page
│── quote.html       ← Get Quote page
│── contact.html     ← Contact page
│
├── css/
│   └── style.css    ← All styles (1200+ lines)
│
├── js/
│   └── script.js    ← All interactivity
│
├── images/
│   ├── logo.png     ← Nasya Cargo logo
│   └── products/    ← Product/cargo images (add here)
│
└── assets/
    ├── fonts/       ← Custom fonts (add here)
    └── icons/       ← Icons & favicon
```

## Usage
1. Open `index.html` in your browser to preview locally.
2. To deploy: upload entire folder to hosting provider.
3. Rename to `index.html` on Netlify or GitHub Pages.
